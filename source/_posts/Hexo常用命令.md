## hexo clean ##

``` bash
hexo clean # 清除缓存文件db.json 和已生成的静态文件public
```
## hexo g ##

``` bash
hexo g # 是hexo generate的简写形式，生成静态文件到public文件夹
```
## hexo d ##

``` bash
hexo d # 是hexo deploy的简写形式，部署静态文件到设定的仓库
```
## hexo clean && hexo g && hexo d ##

``` bash
hexo clean && hexo g && hexo d # 常用的组合命令，清除缓存，生成静态文件，并部署到指定仓库
```
