## 1、安装系统默认的JAVA

```
apt-get install default-jdk
```

## 2、安装JAVA11或JAVA13

```
add-apt-repository ppa:linuxuprising/java
apt-get update
apt-get install oracle-java11-installer
apt-get install oracle-java13-installer
```
