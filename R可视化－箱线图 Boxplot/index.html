发表于 2020-02-07 | 分类于 编程语言 | | 热度 °C

关于箱线图的数据类型：
X轴为定性变量，如本例中的A，B，C，D；Y轴为定量变量，即具体的数值；箱线图可以用来观察数据的分散情况，以及是否有异常值；因此，本例中使用rnorm()函数产生的正态分布数据作为输入；而对于本例中的分类C，明显由两组数据组成，绘制小提琴图或许是更直观的方式；
