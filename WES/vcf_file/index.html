发表于 2020-11-18 | 分类于 编程语言 | | 热度 °C

VCF（Variant Call Format），是存储变异信息的文件格式，包括SNP（单核苷酸多态性，指单碱基变异）、INDEL（插入或缺失，指短片段变异）、SV（结构变异，指长片段变异）、CNV（拷贝数变异）。
