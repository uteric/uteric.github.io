R可视化-棒棒糖图

发表于 2020-02-11 | 分类于 编程语言 | | 热度 °C

棒棒糖图，介于柱形图与散点图之间，之所以这么说，是因为它能以柱形图与散点图的数据作图，即两列定量变量（数字），或一列定量变量（数字）一列分类变量（字符串）；

有趣的是，棒棒糖图其实不是由一个函数绘制的，而是由两个函数叠加成的，即使用geom_point()绘制点，使用geom_segment()绘制线；
